# Design System Minimal

Base design system pour la migration progressive du site.

- CSS runtime : `assets/css/design-system/`
- Patterns HTML de reference : `design-system/components/`
- Strategie : tokens nommes + classes `c-`, `l-`, `u-` + migration scopee via `data-ui`

Le design part de la capture de home et du site existant :

- fond marine dense
- surfaces claires / verre leger
- accent orange chaud
- secondaire bleu atelier
- cards arrondies, ombres larges, contrastes nets
