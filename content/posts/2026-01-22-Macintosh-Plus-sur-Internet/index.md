## Matériel réel

### BlueSCSI
- Émulation disque dur SCSI
- Stockage SD
- Support Pico / Pico W (selon version)
- Base idéale pour préparer un environnement réseau

## Accès à ChatGPT (réaliste)

Rappels importants :
- Pas de HTTPS direct
- Pas de navigateur moderne
- Obligatoire :
  - Proxy HTTP simplifié
  - Passerelle locale
  - Interface texte

Objectif :
- **Envoyer du texte**
- **Recevoir du texte**
- Pas d’interface graphique moderne
